# Remove the duplicate elements from the given list in constant space and O(n)

## Constraints
* Do not use any extra space
* Solve the problem in O(n) time complexity

## Sample Input Output

**Input**
[2,3,3,4,5,6]
**Output**
[2,3,3,4,5,6]

**Input**
[2,4,4,4,6,7,7,7,7,8]
**Output**
[2,4,4,6,7,7,8]

**Input**
[2,2,2,2,2,2]
**Output**
[2,2]
